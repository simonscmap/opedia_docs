How to Contribute
-----------------

You can contribute directly to Simons CMAP via the Github_ page or you can make suggestions via email or Slack_.

How to suggest new datasets set to be added to database
-------------------------------------------------------

If you wish to suggest a dataset be added to the database or have some data that you would like added, please email Raphael (nrhagen@uw.edu) or Mohammad (mdehghan@uw.edu)
