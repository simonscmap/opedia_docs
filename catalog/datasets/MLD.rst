Dataset Description
*******************

**Mixed-Layer-Depth** is a model derived ocean surface product collected from the Operational Mercator Global Ocean Analysis and Forecast System.
The data has 1/12째 degree horizontal resolution with regular longitude/latitude intervals and an equirectangular projection.
